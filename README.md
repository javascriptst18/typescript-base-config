## Installation

1. Clone this repo:
```bash
git clone https://github.com/javascriptst18/typescript-base-config
```
2. `cd` into folder:
```bash
cd typescript-base-config
```
3. Install `typescript` globally via NPM:ยง
```bash
npm install -g typescript
```
4. Run the following command from the terminal:
```bash
npm run dev
```
